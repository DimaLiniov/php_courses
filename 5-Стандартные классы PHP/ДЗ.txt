Стандартные классы PHP

1. Классы для работы с датой и временем

Создайте две случайные даты (с помощью mt_rand()) и установив их в 2 объекта класса DateTime.
Найдите количество дней, часов, минут и секунд, на которое одна дата отличается от другой.
Сделайте разбиение на периоды с интервалом 1 день.
В цикле выведите все даты, которые содержатся между созданными в 1-м пункте, с интервалом из 3-го пункта.

2. Итераторы

Создайте класс, реализующий интерфейс Iterator.
Добавьте 2 свойства: целое число index (будущий индекс массива) и обычный список list с несколькими элементами.
В методах из интерфейса Iterator работайте со свойством index и возвращая при необходимости return $this->list[$this->index];
В цикле переберите объект и выводите каждый раз текущее значение в цикле. Убедитесь, что вывелось в итоге всё содержимое свойства list.
Примечание: Например, в rewind() Вы должны написать $this->index = 0; В current() - return $this->list[$this->index] и так далее.

3. Виртуальные массивы

Допишите класс, созданный в предыдущем упражнении, реализовав интерфейс ArrayAccess.
Необходимо, чтобы можно было легко и быстро добавлять в список $list (свойство из предыдущего упражнения) новые элементы.
Добавьте несколько элементов в объект, обращаясь к нему как к обычному массиву.
Убедитесь, что перебор списка за счёт интерфейса Iterator по-прежнему работает без ошибок.

4. Класс Directory

Выберите произвольную директорию у себя на компьютере, в которой много файлов и вложенных директорий.
Используя класс Directory и рекурсию, выведите дерево выбранной директории.

5. Класс Generator

Напишите функцию-генератор, которая будет принимать 3 параметра: $start, $to, $step.
Функция должна на каждой итерации выводить текущее значение $i из цикла for ($i = $start; $i < $to; $i++)
Переберите этот генератор.
Вызовите метод send, в котором передайте целое число.
В генераторе надо принять это целое число и установить его в step.
Убедитесь, что при изменении значения step, шаг в цикле действительно изменился.

6. Класс IntlChar

Выведите таблицу, где в первой колонке будет выводиться код символа, а во второй колонке его вид.
Примечание: Безусловно, нужно использовать класс IntlChar и цикл. Количество итераций необходимо установить экспериментально. Сама таблица должна быть выведена через теги <table>, <tr> и <td>.

7. Отражения

Создайте класс Point с двумя свойствами x и y, конструктором и 4 методами (getX(), getY(), setX($x) и setY($y)).
Используя Reflection, выведите следующую информацию о созданном классе:
«Имя класса: Point
Свойства: x, y
Методы: getX(), getY(), setX($x), setY($y)»
Примечание: Безусловно, здесь потребуется использование циклов.
